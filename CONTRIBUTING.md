# Contributing to OPEN GATE

## Developer Certificate of Origin (DCO)

All contributions must include a Signed-off-by line indicating agreement to the DCO:

```
Signed-off-by: John Doe <john@example.com>
```

Use `git commit -s` to automatically add this.

## Code Style

- **C**: C17, MISRA-C 2012 guidelines
- **SystemVerilog**: 2017 standard, lowercase_with_underscores
- **Python**: Black formatting, type hints
- **Rust**: 2021 edition, clippy clean

## Pull Request Process

1. Fork the repository
2. Create feature branch (`git checkout -b feature/amazing`)
3. Commit changes with DCO (`git commit -s`)
4. Push to branch (`git push origin feature/amazing`)
5. Open Pull Request

## Security Issues

Email security@open-gate.io with PGP encryption (key: 0xA1B2C3D4).
24-hour response target.
